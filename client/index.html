HELP MENU
  • Filtering your library by category
    Your library possesses Categories list, located in the upper-left corner. From it you can select the desired categories to show in the library for easy search.
  • Filtering your library by text search
    Next to the Categories list you will see an input field. You can use it to find matches in the name and the description of your records. The filed has the functionality of a search as you type, so it is not obligatory to press the search button.
  • Edit a place in your library
    Every record in your library can be edited. In the example above, you will see how to navigate to the edit menu and save the desired changes.
  • Delete a place from your library
    Follow the steps in the upper example to delete a place from your library. Warning: once delete a record cannot be recovered!
  • Search controls
    In upper part of the Search place menu you will see a few controls: Create place button (for creating a new place), type of search dropdown (types of search are: key-word, coordinates, sentence and address), input for query (text to search for), Search button (initiate search and show results) and all of the search engines in which to search (disabling any of them will result in less results).
  • Key-word search
    To search using a key-word, select "Key-word" from the dropdown, enter your query and press search.
  • Sentence search
    To search using a key-word, select "Sentence" from the dropdown, enter your query and press search.
  • Coordinates search
    To search using a key-word, select "Coordinates" from the dropdown, enter your coordinates (latitude and longitude) and press search.
  • Address search
    To search using a key-word, select "Address" from the dropdown, enter your address (City, street, number, etc.) and press search.
  • Create a new place
    If you are not able to find the desired place in any of the connected databases (Google Places, Here Places or Sygic) you can add it yourself. Press the Create place button and fill in the fields to add a record to your library.
